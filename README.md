## CUESTIONARIO
- ¿Por qué Git y GitHub son herramientas importantes para el curso?
Nos permite trabajar remotamente un proyecto sea grupal o individual
- ¿Qué conductas éticas deberían promocionarse cuando se usa un Sistema de Control de Versiones?
Tener una mejor comunicacion para los proyectos
Respetar ideas o publicaciones
